Vue CLI

1.2. eslintc 全局变量、忽略规则配置

1.3. webpack（build、config文件）

  • 修改host (便于访问、演示)

  • webpack配置proxy (本地解决跨域，域名前缀问题)

  • 如果你的前端应用和后端 API 服务器没有运行在同一个主机上，你需要在开发环境下将 API 请求代理到 API 服务器。这个问题可以通过 vue.config.js 中的 devServer.proxy 选项来配置。

  • 修改alias （便捷的路径访问）
