### 一、安装TypeScript
#### 1.1 通过npm（Node.js包管理器）
```
    npm install -g typescript
```
#### 1.2 安装Visual Studio的TypeScript插件 `ms-vscode.vscode-typescript-tslint-plugin`
### 1.3 编译代码
```
    tsc xxx.ts
```
